<p align="center">
  A modern macOS terminal emulator with built-in Node.js version management.<br>
  No more switching between terminal and nvm - manage your Node.js versions directly from your terminal.

## System Requirements

- macOS 10.15 (Catalina) or later
- Apple Silicon (M1/M2/M3) or Intel processor

## Quick Start

1. Download the appropriate DMG for your Mac
2. Open the DMG and drag nterm to Applications
3. Launch nterm and start using Node.js with built-in version management
